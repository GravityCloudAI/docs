## AI Code Review Configurations

Matter AI offers several configuration options to customize your AI code review experience. These settings allow you to tailor the AI review process to your team's specific needs and workflow preferences.

### Available Configurations

| Configuration | Description |
| ------------- | ----------- |
| Enable PR Review Comments | Allows AI to add inline comments on specific code sections in pull requests |
| Enable PR Review on commits | Triggers AI code review automatically when new commits are pushed |
| Enable PR Description Updates | Allows AI to update the PR description with review findings |
| Enable Quality Recommendations in Summary | Includes code quality improvement suggestions in the review summary |
| Enable RuleSets in Code Review | Applies custom rule sets to guide the AI review process |
| Enable PR Description as Comment | Posts the PR description as a comment for better visibility |
| Enable Sequence Diagram in Summary | Generates and includes sequence diagrams in the review summary for better visualization of code flow |
