## Cost Optimisations - Recommendations

<Note>Save upto 40% cloud costs per month</Note>

Cost Recommendations is a real-time system that keeps a check on deprecated, idle or older generation cloud resources that can help reduce costs such as:
- Older generation storage and compute
- Over provisioned resources
- Idle resources not in use
