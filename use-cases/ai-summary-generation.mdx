## AI Summary Generation

<Note>Save 100% time with auto-generated PR descriptions and checklist</Note>

Matter AI analyzes your pull requests and generates a perfect PR description and auto-fills the PR checklist.

### PR Description and checklist

1. Create a new PR and leave the description empty or the empty checklist
2. Matter will automatically generate the PR description and if the checklist exists, it will auto-fill the checklist
3. Any further commits will also auto-update the PR description and checklist
