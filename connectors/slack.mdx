## Overview
Adding Slack webhooks to MatterAI enables Code Quality and Vulnerability notifications on merge to your Slack channels.

## Steps

1. Visit https://app.matterai.dev/settings and enter your Slack Webhook URL under `PR Merge Alerts`

<img
  className="block"
  src="/images/connectors/slack-webhooks.png"
  alt="Matter AI Connectors"
/>

2. Configure the thresholds for Code Quality and Vulnerability notifications.
- Code Quality Percentage: If a PR is merged below this percentage, it will be notified in Slack.
- Package Vulnerability: If a PR is merged with an existing package vulnerability, it will be notified in Slack.
