## Overview
Installing Gravity Cloud's official Github App enables CI/CD for your services

## Steps

1. Visit https://console.gravitycloud.ai/connectors and then click on Connect under Github.

<img
  className="block"
  src="/images/connectors/connectors.png"
  alt="Gravity Cloud Connectors"
/>

2. On the Github App page, select `Install` or `Configure`

<img
  className="block"
  src="/images/connectors/github-app.png"
  alt="Gravity Cloud Connectors"
/>

3. Select the organisation or specific repository to install the app to. You can update this list later via the same steps.

<img
  className="block"
  src="/images/connectors/github-app-repo.png"
  alt="Gravity Cloud Connectors"
/>

<Note>Once installed, Gravity Cloud will pull the list of repositories. Gravity does not store any code, the code is only fetched for CI/CD runtime and then purged from an isolated VM</Note>
