## What is Gravity Cloud?

<Note>Simplify DevOps Chaos & Management to save your team's time, effort and cloud costs!</Note>

Gravity Cloud is a Unified DevOps Platform that provides internal tools for building, managing and scaling cloud infra without needing to build any internal IDP.

Gravity supports multiple clouds as well as multiple workload management such as AWS ECS, Google Cloud Run, Kubernetes and many more.
